# Guia de Recomendações sobre Filmes de Fantasia

Este projeto é uma página criada para recomendar filmes do gênero cinematográfico **fantasia**, apresentando histórias que misturam magia, aventura e mundos extraordinários. A página indica ao leitor/espectador três obras importantes do gênero: **Wicked**, **a saga Harry Potter** e **Alice no País das Maravilhas (2010)**, destacando os elementos marcantes de cada filme e incentivando o público a conhecê-los.

## Tecnologias usadas
- **HTML5**
- **CSS3**
